## 신규 기능

### API
- GET /api/finance/bills/[id]/details
  - 기본 수강료, per-lesson 출결 정보, EnrollmentRule (Layer 2), MonthlyAdjustment (Layer 3), 레거시 조정, 최종 청구액

### 응답 보강
- Bill 매핑에 hasAdjustments 필드 추가
  - per-lesson: base = chargeable * fee
  - monthly: base = class.fee
  - bill.amount !== base 또는 adjustAmount > 0 면 true
- Bill 매핑에 baseFee 필드 추가 (반의 기본 단가)

### UI
- BillingTab 청구액 셀에 [세부 내역] 버튼 신설
  - 조정 있음: 보라색 활성, 클릭 시 모달
  - 조정 없음: 회색 비활성, 한 눈에 구분
- BillDetailModal: 기본료 -> 조정 항목들 -> 최종 청구액 breakdown
- 자동 형제할인은 [자동] 배지, 월별 조정은 [월별] 배지로 시각 구분
